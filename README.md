## Requirement

* PHP >= 7.2

## Installation

``` bash
$ composer require guanguans/laravel-exception-notify --prefer-dist -vvv
```

## Usage

1. replace `guanguans/laravel-exception-notify` -> `vendorName/package-name`
2. replace `Guanguans\\LaravelExceptionNotify` -> `VendorName\\PackageName`
3. replace `Guanguans\LaravelExceptionNotify` -> `VendorName\PackageName`
4. replace `ityaozm@gmail.com` -> `your email`
5. execute `$ composer dumpautoload`

## Testing

``` bash
$ composer test
```
